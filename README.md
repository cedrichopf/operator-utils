This repository contains a Go module with utilities to build Kubernetes Operators using the [Operator SDK](https://sdk.operatorframework.io/).

## Requirements

- Operator SDK
- Go (Version >=1.20)

## Usage

Install the Go module:

```sh
go get github.com/cedrichopf/operator-utils
```
